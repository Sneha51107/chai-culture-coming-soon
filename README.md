# Chai Culture – Coming Soon Landing Page ☕

A responsive "Coming Soon" landing page created for a tea brand concept called **Chai Culture**.
Users can enter their email address to get notified when the website is launched.

## Features
- Coming soon launch UI
- Email input field for notifications
- Clean and minimal design
- Responsive layout

## Tech Stack
- HTML
- CSS (Plain CSS)
- Javascript

## What I Learned
- Building responsive layouts using plain CSS
- Styling forms and buttons
- Working with flexbox and basic layouts
- Improving UI design with CSS
